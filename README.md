## Descrição

Este projeto consiste na criação de um laboratório de segurança onde aplicarei meus conhecimentos em redes. 

Ele incluirá a implementação de máquinas de servidor, cliente e firewall, todas configuradas para demonstrar o funcionamento e a segurança de uma infraestrutura de rede. 

O objetivo é explorar e evidenciar práticas de segurança, além de entender melhor a interação entre os diferentes componentes dentro de um ambiente de rede protegido.
